# max_density_kanji
## 概要
文字の黒い部分が一番高い文字を求めるプログラム。

## 結果
「麤」

### 環境
- OS: Windows11
- font：YuGothR
- 検査対象：Unicode文字の内、説明に'CJK'といった文字列が含まれる場合
